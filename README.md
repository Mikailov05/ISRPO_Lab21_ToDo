## Описание (что изучили)

В ходе выполнения лабораторной работы было создано веб-приложение "TODO-лист" (список задач). В процессе разработки были изучены и применены следующие технологии и концепции:

- **Работа с DOM (Document Object Model):**
  - Поиск элементов с помощью `getElementById()`
  - Создание элементов через `createElement()`
  - Добавление и удаление элементов из DOM-дерева с помощью `appendChild()` и `remove()`
  - Настройка CSS-классов через свойство `className`
  - Изменение текстового содержимого с помощью `textContent`

- **Обработка событий:**
  - Добавление обработчиков с помощью `addEventListener()`
  - Обработка кликов на кнопках "Выполнено" и "Удалить"
  - Обработка нажатия клавиши Enter в поле ввода
  - Событие `DOMContentLoaded` для инициализации приложения

- **Управление состоянием приложения:**
  - Глобальная переменная-счётчик задач `totalTasks`
  - Функция `updateCounter()` для синхронизации интерфейса с состоянием

- **CSS-стилизация:**
  - Сброс стандартных отступов и применение `box-sizing: border-box`
  - Flexbox для расположения элементов
  - Псевдоклассы `:hover` и `:focus` для интерактивности
  - Стилизация состояний элементов (зачёркивание выполненной задачи)
